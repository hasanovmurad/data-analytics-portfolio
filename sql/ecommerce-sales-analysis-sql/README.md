# E-Commerce Sales Analysis with SQL

## Project Overview

This project analyzes sales data from an e-commerce business using SQL.

The goal is to answer common business questions related to revenue, customers, products, and sales performance.

## Business Questions

* Which region generated the highest revenue?
* Which product category performed best?
* Who are the top customers?
* What is the average order value?
* How many orders were placed?
* How does revenue change over time?

## SQL Concepts Used

* SELECT
* GROUP BY
* ORDER BY
* SUM
* AVG
* COUNT

## Files

* queries.sql – SQL queries used for analysis
